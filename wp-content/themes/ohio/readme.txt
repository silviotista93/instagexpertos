Ohio is flexible and high performance creative theme.

Author: Colabrio
Author URI: https://clbthemes.com
Docs Website: https://docs.clbthemes.com/ohio
Help Center: https://colabrio.ticksy.com

== Installation ==
Manual installation:

1. Upload the `ohio` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress;
2. Check Theme Settings menu to change an appearance using specific options;

== Custom Logo ==
This theme supports custom logo for the header part. 
You can change or remove it from Theme Settings -> Branding.
